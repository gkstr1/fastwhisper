# Recruiting

ATS (Applicant Tracking System) notes, candidate information, and recruitment workflows.

## Scorecard ATS

### Core Features
- AI-powered recruitment suite
- LinkedIn integration
- Kanban board pipeline
- Automated interview scheduling

### Pipeline Stages
1. **Sourced** - Initial candidate capture
2. **Initial Screening** - First contact and qualification
3. **Technical Assessment** - Skills evaluation
4. **Interview Center** - AI-generated questions + scorecards
5. **Final Interview** - Decision meeting
6. **Offer Extended** - Negotiation phase
7. **Hired/Rejected** - Final outcome

## Automation Workflows

### Interview Center Trigger
When candidate moves to "Interview Center":
1. ✅ Resume parsed and stored
2. ✅ AI generates 10 custom questions
3. ✅ Zoom meeting auto-created
4. ✅ Calendar invite sent
5. ✅ Scorecard template prepared
